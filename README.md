# Dream Store

드림시큐리티 사내 AI 도구 마켓플레이스 — **컨셉 시연용 목업**

임직원이 직접 만들거나 발굴한 AI 도구(Skill · MCP · Subagent · 프롬프트)를 등록하고,
다른 임직원이 설치해 사용하고 평가하는 사내 앱스토어입니다.
그 평가 데이터가 AI 활용 우수자 시상의 근거가 됩니다.

- **기획 문서** — [tasks/master-plan.md](tasks/master-plan.md)
- **배포** — GitHub Pages
- **데이터** — Supabase (미설정 시 브라우저 로컬 저장소로 동일하게 동작)

## 화면

| 경로 | 화면 |
| --- | --- |
| `#/` | 랜딩 — 전사 절감 시간, 인기·신규 도구, 시상 후보 TOP 5 |
| `#/store` | 스토어 — 직무·유형·에이전트·구분 필터, 정렬, 검색 |
| `#/tool/:id` | 도구 상세 — 사용 사례, 설치 가이드, 후기, 종합 점수 |
| `#/review/:id` | 후기 작성 — 구조화 양식, 월 절감 시간 실시간 계산 |
| `#/submit` | 도구 등록 — 자체 제작 / 큐레이션 트랙 선택 |
| `#/setup` | 데이터 연동 설정 (운영용) |

## 로컬 실행

정적 사이트라 빌드가 필요 없습니다. 모듈 스크립트를 쓰므로 `file://` 이 아닌 HTTP 로 열어야 합니다.

```bash
python3 -m http.server 8000
# http://localhost:8000
```

## Supabase 연동

1. `supabase/schema.sql` 을 Supabase SQL Editor 에서 실행
2. `assets/js/config.js` 의 `SUPABASE_URL`, `SUPABASE_ANON_KEY` 입력
3. 사이트에서 `#/setup` → **시드 데이터 업로드** 클릭

설정을 비워두면 시드 데이터와 브라우저 저장소로 동작하며, 화면과 기능은 완전히 동일합니다.

## 평가 체계

종합 점수 100점 = **사용 실적 40** + **동료 평가 30** + **거버넌스 심사 30**

상금이 걸린 평가는 왜곡되기 쉬우므로, 조작이 어려운 사용 실적(특히 재사용률)을 중심에 두고
운영위원회 심사를 최종 방어막으로 둡니다. 자세한 설계는 기획 문서를 참고하세요.
